# Smart Resume Analyzer (ATS System)

A full-stack MERN application that analyzes resumes against job descriptions and provides ATS compatibility score, skill gap analysis, and improvement suggestions.

## Features

- PDF Resume Parsing
- ATS Compatibility Scoring
- Matched vs Missing Skills (Color Coded)
- Actionable Suggestions
- Professional PDF Report Download
- Responsive & Modern UI

## Tech Stack

**Frontend:** React.js, Tailwind CSS, jsPDF  
**Backend:** Node.js, Express.js, Multer, pdf-parse
